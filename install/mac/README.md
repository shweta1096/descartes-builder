# DescartesBuilder for macOS (arm64)

## Launching the App

You can start the application in two ways:

1. **Run directly from the DMG**
   - Open the `DescartesBuilder-macOS.dmg`.
   - Drag `DescartesBuilder.app` to your preferred folder or run it directly from the DMG.

2. **Copy to Applications**
   - Drag `DescartesBuilder.app` from the DMG to your `/Applications` folder.
   - Open `DescartesBuilder.app` from `/Applications` like a normal Mac app.

## Contents of the DMG

- `DescartesBuilder.app` - the main application bundle
- `examples/` - example files to get you started
- `README.md`

## Notes

- This version is for **arm64 Macs (Apple Silicon)** only.
- If you encounter security warnings when first opening, macOS may block unverified apps:
  - Open Settings -> Privacy and Security -> Scroll down to see DescartesBuilder. Click on 'Open Anyway' 
  - Try to launch the app again, it should run.
